What is event delegation in the DOM?

Event Delegation is a useful method which allows you to handle a specific element to a parent element. we can target specific elemnet by event delegation. We use event.target for target.

What is event bubbling in the DOM?

Event bubbling is a method in javascript DOM where an event start form the target element and then bubbles to the up. We can stop it using event.stopPropagation();
